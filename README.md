# [identiForm Website](https://identiform.com/)

## Install

```bash
npm i
```

## Run

```bash
npm start
```

## One click deployments

Initially:

```bash
chmod +x pre_local.sh
./pre_local.sh
```

Then any push will be:

```bash
git push production master
```
